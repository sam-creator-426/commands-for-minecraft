Explore the most useful and fun Minecraft commands categorized for different purposes.

General Commands

  • /give - Give items to a player: /give [amount]
  • /tp - Teleport to other players or coordinates: /tp
  • /time set - Set the in-game time: /time set
  • /weather - Change the weather: /weather [duration]
  • /summon - Summon entities or mobs: /summon [position]
  • /spawnpoint - Set a player’s spawn point: /spawnpoint [x] [y] [z]
  • /effect - Apply effects like speed, strength, etc.: /effect give [duration] [amplifier]
  • /clear - Clear items from a player’s inventory: /clear [item] [amount]
  • /locate - Locate structures like villages: /locate
  • /help - Get help on a command: /help

World Editing Commands

  • /fill - Fill a region with blocks: /fill
  • /clone - Clone a section of the world: /clone [destination]
  • /setworldspawn - Set the world spawn: /setworldspawn [x] [y] [z]
  • /worldborder - Set the world border size: /worldborder set
  • /particle - Create a particle effect: /particle [pos] [speed] [count]
  • /gamemode - Change the game mode: /gamemode [player]
  • /difficulty - Change the game difficulty: /difficulty
  • /gamerule - Modify game rules: /gamerule
  • /summon - Summon a mob at a specific location: /summon [x] [y] [z]
  • /execute - Execute commands as other entities: /execute
